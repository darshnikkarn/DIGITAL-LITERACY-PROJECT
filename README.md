Project Overview
This repository contains my Digital Literacy Portfolio, created as a Student Digital Ambassador. The purpose of this project is to help peers navigate the digital world effectively, from building a professional online presence to practicing safe communication and recognizing online threats. All tools used for this project are free and browser-based.

Module 1: Digital Literacy Awareness Infographic 
Task: Creation of a visual resource covering safe internet practices, useful tools, and the importance of digital literacy.
Link: task-1-presentation/

Module 2: Build Your Student Digital Portfolio 
Task: Establishment of a foundational professional presence across platforms like GitHub, LinkedIn, Kaggle, Stack Overflow, or ResearchGate.
Link: task-2-portfolio/ 

Module 3: Explore Coding & Collaboration Platforms 
Task: Familiarization with coding practice sites (like HackerRank or CodeChef) and the creation of a Google Workspace collaboration tool (Google Form quiz).
Link: task-3-platforms/

Module 4: Professional Email & Etiquette Guide
Task: Drafting professional emails for academic and internship contexts, alongside a Social Media Do's and Don'ts checklist.
Link: task-4-email-etiquette/

Module 5: Cybercrime Awareness Case Study & Prevention
Task: Researching a cybercrime scenario (such as phishing or UPI fraud) and creating a "Stay Safe Online" prevention checklist tailored for college students in India.
Link: task-5-cybercrime/
